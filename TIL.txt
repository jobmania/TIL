  전문가 되기 위해 Git을 정리하자
1. Git이란 버전관리도구 (형상관리도구) 관리시스템
  - 추가적으로 분산형 관리시스템이라고 
  장점 : 병렬 개발 가능, 인터넷 없는 공간에서 개발 가능
2. Git hub란? : 웹호스팅 서비스 쯤?

3. 용어
 repo : repository 저장소 (로컬, 원격)
 commit : 변경된 작업에 대해 확정하고 저장소에 저장(사진찍는느낌?)
 add : 변경한 작업중 선택하여 커밋에 반영할지 안할지 하는 활동
 tracking : repo간 연결
 push : 로컬 >> 원격
 pull : 원격 >> 로컬 (로컬을 나라고 생각하면 편함)
 clone : 원격 repo 복사 붙여넣기
